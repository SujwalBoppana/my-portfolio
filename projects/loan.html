Loan Status Prediction

A predictive web application for estimating loan approval outcomes from applicant features.

Overview

The project applies supervised ML models to classify loan approval likelihood. It includes feature preprocessing, model selection, and a lightweight web UI for prediction input and result delivery.

Tech Stack

Flask Python Machine Learning HTML CSS JavaScript Heroku

Key Highlights

  • Built data preprocessing and prediction flow for loan applications.
  • Exposed model inference through a simple web interface.
  • Focused on practical decision support for financial workflows.
