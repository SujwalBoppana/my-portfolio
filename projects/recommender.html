Recommendation Engine

Movie Recommender System

A collaborative-filtering movie recommendation app that suggests relevant titles based on user preferences.

Overview

The project uses similarity-based recommendation logic to generate personalized movie suggestions. It combines data processing and web app delivery to provide user-facing recommendations.

Tech Stack

Django JavaScript Bootstrap SQLite Pandas AWS S3 Heroku

Key Highlights

  • Implemented collaborative filtering for recommendation generation.
  • Integrated model outputs into a Django-driven web experience.
  • Focused on relevance and response speed for recommendation lookups.
