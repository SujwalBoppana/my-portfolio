Attendance Based on Face Recognition

A facial-recognition attendance workflow designed to reduce manual effort and improve accuracy in attendance tracking.

Overview

The application captures facial data, identifies users, and records attendance through a streamlined interface. It was designed for classroom and institutional usage where rapid and reliable attendance collection is essential.

Tech Stack

Python Machine Learning OpenCV MySQL Tkinter

Key Highlights

  • Automated attendance marking with single-pass facial recognition.
  • Reduced repetitive manual entry for instructors and administrators.
  • Integrated data handling for attendance records and retrieval.
