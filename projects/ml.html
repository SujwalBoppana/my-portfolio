Machine Learning Showcase

ML and DL Web App

A web platform used to present and run machine learning and deep learning demonstrations.

Overview

This application aggregates multiple ML and DL workflows into one interface. It was designed to make experimentation and demonstration easier by exposing models through a Flask backend.

Tech Stack

Flask Python SQLite HTML CSS Heroku

Key Highlights

  • Created a centralized interface for ML and DL demonstrations.
  • Connected backend model flows with user-facing prediction pages.
  • Structured the app for clean extension with additional models.
