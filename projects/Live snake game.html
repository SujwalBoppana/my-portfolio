Live Snake Game using Hand Tracking

A real-time snake game controlled through hand gestures detected via computer vision.

Overview

The game replaces keyboard input with camera-based gesture control. It combines tracking, game-loop logic, and visual feedback for a more interactive user experience and experimentation with real-time CV interfaces.

Tech Stack

Python OpenCV MediaPipe Computer Vision GitHub

Key Highlights

  • Implemented gesture-driven controls for core gameplay actions.
  • Connected camera input with game mechanics in real time.
  • Delivered a playful demonstration of CV-based human-computer interaction.
