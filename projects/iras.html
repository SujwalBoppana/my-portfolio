Cloud and Scalable Systems

Image Recognition as a Service

An elastic cloud system for image recognition that auto-scales based on incoming workload.

Overview

This system was designed to process image-recognition requests using an event-driven architecture on AWS. Compute nodes scale in and out based on queue pressure, enabling efficient resource usage under variable traffic.

Tech Stack

Java Node.js Python AWS EC2 AWS SQS AWS S3

Key Highlights

  • Implemented elastic scale-out and scale-in behavior for recognition workers.
  • Used queue-driven architecture to decouple web and compute layers.
  • Provisioned cloud components for resilient processing pipelines.
